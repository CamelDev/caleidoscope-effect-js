Caleidoscope Effect v1.0.0


Requierments

1. Node.js
2. GTK+
    For Windows 10 x64 you need to download GTK+ x64 libraries to c:\GTK directory.
    In order to use paperjs, please follow "canvas" dependency installation instructions.
    https://www.npmjs.com/package/canvas    

Build instructions

    1. Run global installation script: npm install -g node-gyp
    2. Build project: npm install from source code root directory
    3. Test project: npm start
